# CobaltStrike_Bruter
## 仅用于授权测试
CobaltStrike TeamServer密码爆破工具<br />
本工具修改自：https://github.com/ryanohoro/csbruter<br />

Fixes：修复当线程过多时，无法成功爆破密码问题<br />
Refactored：优化显示提示<br />
Features：当出现连接错误时，可自动重新尝试错误时的密码<br />
